API Description

  • GET /api/productcategories - Get all product categories (Beer, Wine, Rum...)
    Returned JSON data sample: 
    {
  "ProductCategory": [
    {
      "id": 1,
      "name": "Wine"
    },
    {
      "id": 2,
      "name": "Beer"
    },
    {
      "id": 3,
      "name": "Rum"
    }
  ]
}
  • GET /api/recommendproduct/{weekday}/{mood_category_id}/{product_category_id} - Get all recommend product list, currently the weekends (Friday, Saturday, Sunday) return the same result.
    Sample: /api/recommendproduct/0/1/5 - get the recommend product list for Sunday, Wine Down (id = 1), Spirits (id = 5)
    Returned JSON data sample:

  • GET /api/product/{product_id} - Get detail information of a product, can be expanded to more information if necessary
    Returned JSON data sample:

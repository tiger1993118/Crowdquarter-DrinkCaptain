API Description

  • GET /api/productcategories - Get all product categories (Beer, Wine, Rum...)
    Returned JSON data sample: 
    {
  "ProductCategory": [
    {
      "id": 1,
      "name": "Wine"
    },
    {
      "id": 2,
      "name": "Beer"
    },
    {
      "id": 3,
      "name": "Rum"
    }
  ]
}
  • GET /api/recommendproduct/{weekday}/{mood_category_id}/{product_category_id} - Get all recommend product list
    Sample: /api/recommendproduct/0/1/2 - get the recommend product list for Sunday, Wine Down (id = 1), Beer (id = 2)
    Returned JSON data sample: 
    {
  "RecommendProductList": [
    {
      "id": 1,
      "name": "Molson Canadian",
      "volume": "2838",
      "price": "1350"
    },
    {
      "id": 2,
      "name": "Old Milwaukee",
      "volume": "2838",
      "price": "1100"
    },
    {
      "id": 3,
      "name": "Budweiser",
      "volume": "235",
      "price": "473"
    }
  ]
}
  • GET /api/product/{product_id} - Get detail information of a product, can be expanded to more information if necessary
    Returned JSON data sample: 
    {
  "Product": [
    {
      "id": 1,
      "name": "Molson Canadian",
      "volume": "2838",
      "price": "1350",
      "imageURL": "http:\/\/lcbo.com\/assets\/products\/720x720\/0300699.jpg"
    }
  ]
}
